Third-Party Notices for STAR Windows Build
=========================================

This release archive includes prebuilt STAR executables and the MSYS2-MSYS
runtime DLLs required to run them on Windows.

STAR itself is distributed under the MIT License.
Official STAR repository:
https://github.com/alexdobin/STAR


Bundled MSYS2-MSYS runtime DLLs
===============================

The following DLLs are included in this release archive:

  msys-2.0.dll
  msys-z.dll
  msys-gcc_s-seh-1.dll
  msys-gomp-1.dll
  msys-stdc++-6.dll

These DLLs are redistributed unmodified from MSYS2 packages.


1. msys-2.0.dll
---------------

Package:
  msys2-runtime

Description:
  POSIX emulation engine for Windows.

License:
  GPL

MSYS2 package page:
  https://packages.msys2.org/package/msys2-runtime?repo=msys

Source:
  Please refer to the MSYS2 package page and source package for the
  corresponding version of msys2-runtime.

Notes
=====

The bundled DLLs are included only to make the MSYS2-MSYS build of STAR
runnable on Windows systems without requiring users to manually locate these
runtime libraries.

No modifications are made to these DLLs in this release.

For exact license terms, users should refer to the corresponding MSYS2
package pages and upstream source packages.
