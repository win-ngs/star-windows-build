# htslib License Notes

The STAR source tree vendors htslib under:

```text
STAR/source/htslib/
```

This vendored htslib tree does not include a single top-level `LICENSE` file.
License notices are present in the relevant source file headers.

Observed license families in the vendored files include:

- MIT/Expat-style notices for many htslib files.
- BSD-style notices for CRAM-related files.
- Additional file-local notices in individual source files.

Refer to the source file headers in `STAR/source/htslib/` for exact terms.
